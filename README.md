# MiniSyncContext
C# の async/await の裏側を「イベントループ」と「SynchronizationContext」を自作して観察するデモプロジェクト。

「await の内部で何が起きているか？」を理解するための学習用実装です。

## 実行方法

```bash
git clone https://github.com/ledsun/MiniSyncContext.git
cd MiniSyncContext
dotnet run
```
